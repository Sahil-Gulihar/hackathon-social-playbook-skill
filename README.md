# hackathon-social-playbook

A skill for [SagaPad](https://sagapad.com) that generates a personalized, actionable social media playbook for newly submitted Solana hackathon projects.

## What it does

Most teams ship great products at hackathons but fail to tell their story. In crypto, narrative is half the battle. This skill studies what worked for breakout projects from past Colosseum hackathons and turns those patterns into a custom social strategy for any new project.

## Input

Provide the following details about your project:

| Field | Required | Description |
|-------|----------|-------------|
| Project name | ✅ | The name of your project |
| What it does | ✅ | 1–2 sentence description |
| Who it's for | ✅ | Your target user |
| Tech stack | ✅ | Technologies used |
| Team background | Optional | Helps personalize tone and voice |

## Output

- **Core story angle** — the narrative hook that makes your project worth following
- **4-week content plan** — what to post, when, in what format
- **X Spaces & AMAs to join** — specific Solana community events with context on why they matter
- **Engagement strategy** — how to reply, quote-tweet, and insert into existing conversations
- **First 5 posts written out** — ready to copy-paste on day one
- **48-hour quick wins checklist** — immediate actions to get momentum

## How to use

Install on [SagaPad](https://sagapad.com/skills/hackathon-social-playbook) and provide your project details. The skill will generate your full personalized playbook.
